Caso 1
Interpolación Interior — gap entre dos detecciones reales
// t ∈ (0,1) para cada frame intermedio
t = step / (gap + 1)
x = x₀ + t × (x₁ − x₀)     y = y₀ + t × (y₁ − y₀)
Caso 2
Dead Reckoning Adelante — jugador desaparece al final del track
vx = (x_last − x_prev) / dt    vy = (y_last − y_prev) / dt
x_pred = x_last + vx × step    // clamp: max(−5, min(105, x_pred))
Caso 3
Dead Reckoning Atrás — jugador aparece tarde (inicio del track)
vx = (x_second − x_first) / dt_head
x_pred = x_first − vx × step    // proyecta velocidad en sentido inverso
